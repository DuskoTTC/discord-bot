# Dusko The Tax Collector
***Your* High-Performance, Crash-Resistant Discord Music Bot**

'*Dusko*' is a well optimised Discord bot written in Rust. Designed for the lowest possible memory footprint and data persistence.

## Key Features
- **Rust Native**: Low memory footprint and resource efficient runtime.
- **Crash Proof State**: The tracks/queue state lives in Redis (AOF).
- **Lazy Loading**: By default it only fetches the data for the current and next track.
